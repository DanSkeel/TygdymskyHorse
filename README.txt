Исходные данные лежат по адресу http://yadi.sk/d/BgPJ46oG0N4wx .
Пароль: SwItChDetect_2012

После распаковки архива необходимо:
1. Прописать к полный путь к нему в файле data.h
2. Раскомментировать строку в main.cpp
3. Запустить программу. Если всё пройдёт так, как задумано, то в той же папке, в которой лежали файлы train и test, появится новый файл _train_1_percent, который будет содержать около 1% исходной выборки (~27Mb).
4. Закомментировать строку в main.cpp обратно. Это нужно для того, чтобы не ждать каждый раз уйму времени.
5. Если запустить программу, в конце появится цифирька, которая символизирует то, насколько хорошо (примерно) работает алгоритм (см. http://switchdetect.yandex.ru/results).
